# Security Policy

## Supported Versions

Currently supported versions:

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| 0.3.x   | :x:                |
| < 0.3.x | :x:                |

Current policy is only support latest version.

## Reporting a Vulnerability

To report vulnerability, open new Issues ticket on GitHub and use 'bug' label.
